# TPPA

### Requirements

- NodeJS ^v16.13.2
- NPM ^8.4.1
- Express Realtime API

<hr>

### Clone the Repository

```bash
git clone https://gitlab.com/mikesaraus/tppa_spa.git
```

### Install Node Modules

```bash
cd tppa_spa && npm install
```

### Run in Development Mode

```bash
npm run dev
```

### Run in Production Mode

```bash
npm start
```
